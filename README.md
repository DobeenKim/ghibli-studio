<h1>Studio Ghibli Archive</h1>

<h4>A Studio Ghibli film archive built with Next.js and Tailwind CSS.</h4>
<h4>Explore the beautiful world of Studio Ghibli masterpieces and save your favorite films to your personal collection</h4>
<br>
<h4>Tech Stack</h4>
Framework: Next.js 14+ 

Styling: Tailwind CSS

Language: TypeScript

Icons: React Icons 

Data Source: Studio Ghibli API
